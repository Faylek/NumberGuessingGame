## How to Compile and Run
1. **Clone the Repository:**

   ```bash
   git clone https://github.com/Faylek/NumberGuessingGame.git
2. **Install Cmake:**
   for example on mac os:
   ```bash
   brew install cmake
3. **Creating the Folder Build**

   ```bash
   mkdir build

   ```bash
   cd build
4. **Building this application**

   ```bash
   cmake ..

   ```bash
   make
5. **Run the Game**

   ```bash
   ./NumberGuessingGame
